Tensors in PyTorch: Gradient Calculation

To conquer the deep learning landscape, Hercules engages PyTorch to mold tensors like clay. Let's dissect the PyTorch code snippet: 

import torch

x = torch.tensor([[1.0, 2.0], [3.0, 4.0]], requires_grad=True)
y = x.mean()

y.backward()
x.grad

The given code snippet imports the torch library, creates a 2x2 tensor and enables gradient tracking by setting requires_grad to True. We compute the mean of the tensor and store it in the variable y.

Next, we invoke the backward() function to calculate gradients. Finally, we display the gradients associated with the x tensor by calling x.grad.
